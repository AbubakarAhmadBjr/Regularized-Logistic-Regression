## Overview

High‑dimensional data often cause overfitting and poor interpretability in logistic regression. This project systematically compares three regularization techniques—**L1 (Lasso)**, **L2 (Ridge)**, and **Elastic Net**—on three binary classification tasks:

1. **Synthetic high‑dimensional data** (2000 features, 500 samples)  
2. **Breast Cancer dataset** with degree‑2 polynomial features (~500 features)  
3. **Digits dataset** (digit “3” vs. rest, 64 pixel features)

The experiments evaluate prediction accuracy, sparsity (number of non‑zero coefficients), learning curves, ROC curves, and feature importance.  
The final thesis (PDF) and LaTeX source are also included.

---

## Research Questions

1. How do L1, L2, and Elastic Net compare in accuracy and sparsity on synthetic `p ≫ n` data?  
2. Does Elastic Net offer advantages over L1/L2 when polynomial (correlated) features are introduced?  
3. Do regularized models outperform a simple dummy baseline on real‑world image data (Digits)?  
4. What do learning curves and ROC analysis reveal about bias‑variance trade‑offs?

Key Findings
Elastic Net offers the best trade‑off – it matches L2 accuracy while producing sparse models, and handles correlated features better than L1.

Sparsity with negligible accuracy loss – on synthetic data, feature count reduced from 2000 → 83 (96% reduction) with no drop in accuracy.

Regularization prevents overfitting – learning curves show a narrow train‑CV gap; ROC curves give near‑perfect AUC (0.996).

L1 provides interpretable feature rankings – e.g., mean compactness, worst concave points, worst area are top predictors.
